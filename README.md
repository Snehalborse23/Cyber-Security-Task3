
**Password Complexity Checker**
**Organization:** Comillas Negras | **Internship:** Prodigy Infotech  

This project is a **Password Complexity Checker** built in Python that evaluates the **strength and security** of user passwords.  
It analyzes the password based on multiple parameters such as **length**, **uppercase and lowercase letters**, **numbers**, and **special characters** to determine how secure the password is.  

The program provides instant feedback and categorizes passwords as **Weak**, **Moderate**, or **Strong**, helping users create safer and more reliable passwords.
